## Sections

- **CLI Power Tools** — ripgrep, fd, bat, eza, zoxide, fzf, jq, delta
- **Git** — rebase, bisect, worktrees, reflog, `gh` CLI
- **Docker** — build, run, compose, prune, podman
- **AI CLI Tools** — Claude Code, GitHub Copilot CLI, llm, aider
- **Shell & Productivity** — zsh, starship, tmux, dotfiles
- **Networking & Security** — curl, ssh, openssl, nmap
- **Cloud CLI** — AWS, GCP, Azure, kubectl, terraform
- **macOS Power User** — Homebrew, defaults write, system utilities
- **Package Managers** — winget, brew, scoop, nix, mise
- **Node.js & JS Toolchain** — npm, pnpm, bun, npx, tsx

## Local Development

```shell
pip install mkdocs-material
mkdocs serve
```

Open [http://localhost:8000](http://localhost:8000).

## Deploy

Pushes to `main` auto-deploy via GitHub Actions. To deploy manually:

```shell
mkdocs gh-deploy
```
